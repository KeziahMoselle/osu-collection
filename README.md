# osu-no-name

__The cloud for replays__

You can share your replays or just save them privately.

## Features

* Sign up / Log in
* Upload a replay (Need a rewriting)
* Public or Private
* Manage your replays (View,Edit,Delete)
* Download counter
* Add a Youtube link (In edit)
